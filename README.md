# HTML Resume

My professional resume built entirely with **pure HTML** as a learning project to practice HTML fundamentals and semantic markup.

## About

This is a clean, text-based resume created with HTML5 only—no CSS, no frameworks. The project showcases my background as a Computer Science & Engineering student with interests in Data Science and AI/ML.

## View Live

[https://ozenfern.github.io/HTML-Resume/](https://ozenfern.github.io/HTML-Resume/)

## What's Inside

- **Professional summary** and skill set
- **Education timeline** from primary school through B.Tech
- **Projects** including Python automation, seating systems, and PKM workflows
- **Certifications** from online bootcamps
- **Links** to interests and contact information

## Technologies

- HTML5 with semantic markup

## How to Use

1. Clone the repo:
   ```bash
   git clone https://github.com/OzenFern/HTML-Resume.git
   ```

2. Open `index.html` in your browser

## Learning Goal

This project was created to practice:
- HTML5 semantic elements (`<header>`, `<section>`, `<footer>`, etc.)
- Proper document structure
- Building functional web pages with core HTML
